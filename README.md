# e-plantShopping - Paradise Nursery Shopping Application

## Repository
e-plantShopping

## Overview
Paradise Nursery is an e-commerce shopping application for houseplants.

## Features
- Landing page with background image
- Product listing with categories
- Add to Cart functionality
- Shopping cart with quantity controls
- Delete items from cart
- Total cost calculation

## Technologies
- React.js
- Redux Toolkit
- Vite
- CSS3

## How to Run
1. Clone: `git clone https://github.com/KaraboMphafudi/e-plantShopping.git`
2. Install: `npm install`
3. Run: `npm run dev`
4. Open: `http://localhost:5173`
