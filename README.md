# e-plantShopping - Paradise Nursery Shopping Application

## Project Name
Paradise Nursery - Houseplant Shopping App

## Repository
**e-plantShopping**

## Overview
Paradise Nursery is an e-commerce shopping application for houseplants. Users can browse a variety of plants organized by categories including Air Purifying Plants, Aromatic Fragrant Plants, Insect Repellent Plants, Medicinal Plants, and Low Maintenance Plants.

## Features
- Landing page with company introduction and "Get Started" button
- Product listing page with plants grouped by categories
- Add to cart functionality with button disable after addition
- Shopping cart with quantity adjustment (+/- buttons)
- Delete items from cart
- Dynamic cart icon showing total number of items
- Total cost calculation for all items in cart
- Checkout button with "Coming Soon" message
- Continue shopping button to return to products

## Technologies Used
- React.js
- Redux Toolkit (State Management)
- Vite (Build Tool)
- CSS3 (Styling)

## How to Run Locally

1. Clone the repository:
git clone https://github.com/KaraboMphafudi/e-plantShopping.git

2. Navigate to project directory:
cd e-plantShopping

3. Install dependencies:
npm install

4. Start development server:
npm run dev

5. Open browser and go to:
http://localhost:5173
